Car Sprite Generator

How to use:
1. Click "Generate All Sprites" to render all 22 cars as white sprites
2. Click "Download All" to save them as PNG files to your downloads folder
3. Move the downloaded PNGs to img/cars/ in the game folder
4. The game will automatically use sprites instead of Canvas path rendering

For better graphics: Replace any PNG in img/cars/ with a proper side-profile car illustration (transparent PNG, white/light car, facing RIGHT). The game applies paint color tinting automatically.
